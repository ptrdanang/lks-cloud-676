Deskripsi Projek Deployment

Proyek ini adalah implementasi dari modul deploy website GitHub untuk LKS Cloud Computing. Tujuan utama proyek adalah mendemonstrasikan kemampuan:

  • Penerapan fitur dinamis sederhana (API Publik)
  • implementasi kemanan dasar (.gitignore) dan logging (console.log).
